[Gert Drapers](http://blogs.msdn.com/gertd/) has just [announced](http://blogs.msdn.com/gertd/archive/2008/10/27/the-gdr-rc-is-here.aspx "The GDR RC Is Here!") the long awaited RC for the Data Dude GDR.
